概念知识

数组（list）

数组使用中括号[]来表示

数组函数

append() 用来在数组后面添加一个数字或者添加一个数组或者字典。
extend() 用来在数组后面添加一组或一个数字
pop() 数组弹出最后一个元素，即栈的pop
remove() 用来移除第一个出现的指定数字
reverse() 用来数组反向
sort() 用来数组排序
count() 统计某个元素在列表中出现的次数

集合（set）

集合内容无序、但元素不重复。使用大括号{}，无序的意思是不从0开始的栈

集合函数

add() 可以往集合里面添加元素
clear() 清除集合内所有元素
copy() 复制整个集合
difference() 返回一个集合，是A中有的但B中没有的，A-B
difference_update() 根据上面difference()，将A更新为返回的集合
discard() 删除集合中的一个元素，不管这个元素是否存在于集合之中
intersection() 数学意义上的交集，返回交集的集合
intersection_update() 根据intersection()，将A更新为返回的交集
isdisjoint() 如何两个集合没有交集，则返回True
issubset() 如果另一个集合包含此集合则返回True
issuperset() 如果这个集合包含另一个集合则返回True
pop() 删除这个集合的第一个元素，如果集合为空则报错KeyError
remove() 删除这个集合中的指定元素，如果这个元素不存在则报错KeyError
symmetric_difference() 数学意义为A并B-A交B,返回这样的集合
symmetric_difference_update() 根据symmetric_difference()，将A集合改为此函数返回集合
union() 数学意义为A并B，返回这个集合
update() 将此集合更新为A并B

字典（dict）

字典用来储存两两对应的Key-Value也是使用大括号{}，在查看对应Key的Value时使用中括号[]，比如 passwords = {'Andy':123456 ,'John' : 654321}
Andy 对应着 123456

字典函数

clear() 删除字典中所有的元素
copy() 返回字典为此字典，可用于给新的字典快速负值
fromkeys(seq[, v]) 返回一个新的字典，返回的字典中Key为seq中的Key，对应的Value为v中的值，默认为空。其中seq表示已有的字典名，v可以为空，也可以为一个元素，也可以为一个数组，当数组变动时，字典也跟随着变动。
get(key[,d]) 返回指定Key的值，若Key不存在，则返回d，默认为None
items() 返回这个字典里面的所有元素
keys() 返回这个字典里所有的Key
pop(key[,d]) 移除字典中对应的Key的元素，并返回其对应的Value，如果未找到Key，则返回d，若都没有，则报错KeyError
popitem() 随机返回并删除字典中的一对键和值。如果字典为空，则报错KeyError
setdefault(key[,d]) 和get()类似, 但如果键不存在于字典中，将会添加键并将值设为default
update([dict2]) 把字典dict2的键/值对更新到dict里
values() 以列表返回字典中的所有值

元组（tuple）

tuple与list类似，是有序的，但是tuple里的元素不能修改。创建时使用()

元组函数

count(x) 返回对应x元素的个数
index(x) 返回对应x元素的位置
all() 如果元组内元素都为真则返回真
any() 如果元组内任意元素为真则返回真
enumerate(tuple，d) 将元组转换为enumerate类型，若d为空，则默认从0开始依次对元组内的元素标记，d有值则从值的位置开始
len() 计算元组元素个数
max() 返回元组中元素最大值
min() 返回元组中元素最小值
sorted() 返回一个排好序的元组，但元组本身并没有排序
sum() 返回元组内所有元素之和
tuple() 将列表转换为元组

比较运算

==,!=表示两个变量是否相同或不同，is,is not表示两个变量的内存是否相同

索引切片

变量

变量名的规则

1、合理的变量名而不是单一的字母
2、驼峰命名法
3、声明的变量大写
4、不要在变量名中使用符号
5、不要以数字开头

运算符

算术运算符

+ 加 —- 两个对象相加 a + b 输出结果 30
- 减 —- 得到负数或是一个数减去另一个数 a - b 输出结果 -10
* 乘 —- 两个数相乘或是返回一个被重复若干次的字符串 a * b 输出结果 200
/ 除 —- x除以y b / a 输出结果 2
% 取模 —- 返回除法的余数 b % a 输出结果 0
幂 —- 返回x的y次幂 `ab为10的20次方， 输出结果 100000000000000000000 // 取整除 ---- 返回商的整数部分（向下取整）9//2=4,-9//2=-5`

赋值运算符

= 简单的赋值运算符 c = a + b 将 a + b 的运算结果赋值为 c
+= 加法赋值运算符 c += a 等效于 c = c + a
-= 减法赋值运算符 c -= a 等效于 c = c - a
= 乘法赋值运算符 c = a 等效于 c = c * a
/= 除法赋值运算符 c /= a 等效于 c = c / a
%= 取模赋值运算符 c %= a 等效于 c = c % a
= 幂赋值运算符 c = a 等效于 c = c ** a
//= 取整除赋值运算符 c //= a 等效于 c = c // a
